DeepAlpha Copilot is designed for clarity, consistency, and disciplined reasoning—built as a multi-agent system that produces decision-ready equity research.

System overview

DeepAlpha Copilot combines deterministic analytics (scoring + benchmarks) with an agent-based reasoning layer that routes each question to specialized tools. The goal is to make outputs comparable across tickers while remaining adaptive to context and new information. 

Presentation layer (web UI, /chat)
  ↓
Intelligence layer (root agent + specialist sub-agents)
  ↓
Analytics layer (scoring engine + helpers)
  ↓
Data layer (structured + unstructured artifacts, runtime cache)

Agent architecture

At the core is a root orchestrator that interprets intent, delegates to specialist agents, and synthesizes one coherent answer. Specialist agents expose tools for company scoring, news retrieval, sentiment, flow data, and document retrieval.

Root orchestrator

Understands intent, chooses tools, and produces a single integrated response.

Specialist agents

Each agent is narrowly scoped and calls one or more tools against known data sources.

Tool-first design

Tools return structured outputs; the model synthesizes, prioritizes, and communicates tradeoffs.

Reasoning model

DeepAlpha Copilot uses a hybrid approach:

1) Deterministic analytics

A standardized scoring framework produces stable factor scores and recommendation bands—ensuring comparability across companies.

2) LLM synthesis

The model contextualizes results, resolves conflicts between signals, and explains what matters most for the thesis.

Why this matters

The system stays grounded in structured outputs while still producing the kind of prioritization and narrative clarity analysts need to make decisions.

Deep Alpha Score

The Deep Alpha Score (0–10) evaluates companies across seven pillars—business quality, financial strength, sentiment, strategic positioning, leadership, earnings, and technicals. Scores are mapped into a recommendation band and paired with a risk map and monitoring points.

Comparable

One framework across tickers for faster relative analysis.

Actionable

Outputs include thesis, risks, and what to watch next.

Evolvable

New tools and data sources can be added without changing the product surface.

Document intelligence (RAG)

For filing-based questions, the system performs semantic retrieval over document chunks and synthesizes answers grounded in the retrieved passages. This helps users move from “searching” to “reasoning” while still keeping evidence reviewable.

Production design

DeepAlpha Copilot is designed for low-latency responses by keeping real-time queries on a local cache, while using durable storage for persistence and analytics.

Cloud storage

Durable source of truth for production artifacts.

Local runtime cache

Fast reads for the scoring engine and agent tools.

Warehouse (analytics)

Historical analysis and audits without slowing the user path.
