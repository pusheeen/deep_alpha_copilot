Technical notes (optional)

DeepAlpha Copilot uses a hierarchical agent system: a root agent routes requests to specialized agents, then synthesizes a single response.

  • Tool-first design: tools return structured outputs; synthesis focuses on prioritization and explanation.
  • Multi-tool queries: one user request can trigger multiple internal calls (scores + news + filings + sentiment).
  • Graceful degradation: if a tool is unavailable, the system continues with available evidence and communicates limitations.
